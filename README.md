# tsconfig
Pedro's Shared [TypeScript config](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html)

## Install

```
$ npm install --save-dev @1pedro/tsconfig
```

## Usage

`tsconfig.json`

```json
{
	"extends": "@1pedro/tsconfig",
	"compilerOptions": {
		"outDir": "build",
		"target": "es5"
	}
}
```
